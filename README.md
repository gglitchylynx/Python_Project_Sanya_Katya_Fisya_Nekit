# Autonomous Robot Navigation: Gazebo + OpenCV (PID Control)

Реализация системы автономной навигации мобильного робота с **дифференциальным приводом** в среде Gazebo. Проект использует алгоритмы компьютерного зрения для анализа окружения и PID-регулятор для управления движением.

## Обзор системы
Робот обрабатывает видеопоток с фронтальной камеры в реальном времени, вычисляет отклонение от заданного маршрута и корректирует курс. Использование PID-контроллера обеспечивает плавное маневрирование и стабильность системы при движении на различных скоростях.

### Ключевые характеристики:
* **Тип управления:** Дифференциальное (Differential Drive).
* **Контроллер:** PID (Пропорционально-интегрально-дифференциальный).
* **Middleware:** ROS 2 Jazzy Jalisco.
* **Зрение:** Цветовая сегментация (HSV) и анализ центроида через OpenCV.

## Логика работы

### 1. Обработка сенсоров и зрения
Основная обработка данных происходит в узле `sensor_data_proc.py`:
* **OpenCV Integration:** Конвертация RAW-изображений из симуляции в формат OpenCV.
* **Image Processing:** Бинаризация и фильтрация для выделения целевого пути.
* **Error Calculation:** Определение смещения центра пути относительно центра кадра (ошибка навигации).

### 2. Управление и PID
На основе полученной ошибки рассчитываются команды в топик `/cmd_vel`:
* **P (Proportional):** Мгновенная реакция на отклонение.
* **I (Integral):** Компенсация статической ошибки и дрейфа.
* **D (Derivative):** Демпфирование колебаний при резких изменениях траектории.
* **Lidar Logging:** Параллельное логирование данных с лазерного дальномера через `lidar_logger.py` для анализа окружения.

## Технологический стек
* **Язык:** Python 3.3+
* **Симулятор:** Gazebo (Ignition/GZ)
* **Связь:** `ros_gz_bridge` для обмена данными между ROS и Gazebo.
* **Моделирование:** XACRO (модульное описание робота).
